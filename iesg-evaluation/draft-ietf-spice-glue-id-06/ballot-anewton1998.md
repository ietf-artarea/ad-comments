## Comments

I have read the mailing list discussions on new identifier systems restricting allowed characters to non-ASCII.
It is my opinion that the working group has established the reasons for this decision, and that changes to
this document to accommodate non-ASCII characters would require significant changes to this specification for
unknown benefit.
